roadmap

learn what's coming up next in alpmbuild, or go back to home.

libalpm integration

alpmbuild will be able to check what packages you have installed on your system and which ones you need to install when building a spec file. in true alpmbuild error reporting fashion, it'll also be able to recognise typos in package names and suggest corrections for you.
coming in: next release

reproducibility help

alpmbuild will be equipped with tools and checks to help you build your packages in a reproducible manner, and will help warn you about conditions that may interfere with reproducibility.
coming in: two releases
